**How to use it ? :**

There's some few things that you need to know.
First thing first : the keys words
there are 2 new then C++ :
  - "var" for initialazing a variable
  - "func" for the functions

They don't need to have a type on them (more efficient).
For "func", you can say the return after the parenthesis with "->"

There's also the classes that are diffrent : you need to use ":" for use a member of a class like :
- "a:method()"
or
- "a:variable"
